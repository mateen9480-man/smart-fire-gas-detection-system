# Smart Fire Gas Detection System

## Overview
The Smart Fire Gas Detection System is designed to monitor gases and provide alerts in case of fire hazards. It leverages advanced sensors and real-time data processing for effective safety measures.

## Features
- Gas detection for multiple types of gases
- Real-time alerts and notifications
- User-friendly interface
- Data logging for analysis

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mateen9480-man/smart-fire-gas-detection-system.git
   ```
2. Navigate to the project directory:
   ```bash
   cd smart-fire-gas-detection-system/backend
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
- Start the server:  
  ```bash
  npm start
  ```  
- Access the application at `http://localhost:3000`.
