# Smart Fire and Gas Detection System

## Introduction
This firmware is designed for a Smart Fire and Gas Detection System that helps in identifying dangerous situations and alerting users.

## Features
- Real-time monitoring of gas levels.
- Fire detection using temperature sensors.
- Alerts via push notifications to connected devices.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mateen9480-man/smart-fire-gas-detection-system.git
   ```
2. Navigate into the project directory:
   ```bash
   cd smart-fire-gas-detection-system
   ```
3. Upload the firmware to your device using the appropriate method for your microcontroller.

## Usage
- Make sure the sensors are connected as per the schematic.
- Power the device and monitor the alerts on your application.
