# Smart Fire Gas Detection System

## Frontend Documentation

This documentation provides an overview of the frontend structure and functionality of the Smart Fire Gas Detection System.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mateen9480-man/smart-fire-gas-detection-system.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd smart-fire-gas-detection-system/frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Usage

To start the frontend application:

```bash
npm start
```

This will run the app in development mode.

### Features

- Real-time gas detection monitoring
- User-friendly interface
- Responsive design

- React.js
